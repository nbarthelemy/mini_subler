# MiniSubler

A lightweight Ruby wrapper around [Subler](http://code.google.com/p/subler/)'s CLI interface.

## Installation

Add this line to your application's Gemfile:

    gem 'mini_subler'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mini_subler
